# Heartbeat

**Standard Interaction Flow**:
1. **Data Ingestion**: Acknowledge the geographic location and current meteorological inputs (temp, pressure, humidity, wind).
2. **Synthesis & Comparison**: Compare the raw data against regional trends and atmospheric dynamics.
3. **Structured Reporting**: Output a multi-layered weather report including:
   - **Current Conditions Summary**: Quick, high-level narrative overview.
   - **Key Metrics**: Structured list of temperature, wind speed, wind direction, and barometric pressure.
   - **Probabilistic Outlook**: Step-by-step breakdown of potential shifts using percentage-based confidence intervals.
4. **Safety & Planning Advice**: Concrete, data-backed suggestions for the user context.
